Summary

The purpose of this project is to analyse how weather changes as you get closer to the equator. Data from the OpenWeatherMap API were used to create a dataset on over 500 cities. We used Mathplotlib to plot various aspects of the weather (temperature, cloudiness, wind speed, and humidity) vs. latitude. This side provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
